Come installare il plugin di backup All-in-One WP Migration.

  • Metodo online
    1. Dalla pagina o pannello di controllo di amministrazione di WordPress di Altervista o Pantheon,
      recarsi nel menu laterale e passare sopra la voce Plugin e selezionare la funzione Aggiungi nuovo.
    2. Nella casella di ricerca plugin cercare il plugin All-in-One WP Migration.
    3. Poi cercare il programma All-in-One WP Migration e premere sul pulsante Installa adesso.
    4. Quando l'installazione finisce premere sul pulsante Attiva per attivare il plugin.
  • Metodo offline
    1. Scaricare dallo store online di Wordpress il plugin All-in-One WP Migration .
      (Oppure chiedi al prof. se potrebbe passarti i file dei plugin di backup )
    2. Dalla pagina o pannello di controllo di amministrazione di WordPress di Altervista o Pantheon.
      Recarsi nel menu laterale e passare sopra la voce Plugin e selezionare la funzione Aggiungi nuovo.
    3. Vicino al titolo Aggiungi plugin premere sul pulsante Carica Plugin.
    4. Premere il pulsante Scegli file e selezionare il file ZIP del plugin scaricato prima.
    5. Poi premere su Installa adesso ed attendere che l'installazione vada a buon fine.
    6. A fine installazione dal menu laterale sotto la voce Plugin selezionare la funzione Plugin installati.
    7. Attivare il plugin All-in-One WP Migration.

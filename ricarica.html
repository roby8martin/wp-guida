Come ripristinare Xampp della lezione precedente.

  • Recuperare Xampp dal cestino
    1. Recarsi nella stessa postazione della scorsa lezione.
    2. Aprire il cestino e ripristinare la cartella Xampp.
  • Nel caso non si ha Xampp nel cestino
    1. Ripetere questa procedura poi ritornare in questa sezione per ripristinare il backup.
    2. Segui la procedura del ripristino del tema

Come ripristinare il backup della lezione precedente.

Ripristinare il file tramite il plugin All-in-One WP Migration.
  1. Dal NAS copiare il file di backup creato la scorsa lezione (qui la guida) nel desktop del computer.
  2. Dalla pagina o pannello di controllo di amministrazione di WordPress http://localhost:8081/wp-admin/.
    Recarsi nel menu laterale e passare sopra la voce All-in-One WP Migration e seleziona la funzione Importa.
  3. Nella schermata che appare premere su Importa Da poi selezionare File.
  4. Attendere che il plugin elabori il file di backup.
  5. Ad un certo punto viene fuori un popup che indica che tutti i file e database presenti nel sito verranno sovrascritti.
    Rispondere al popup premendo su Procedi >.
  6. Riparte un altro caricamento finale dove sta scrivendo tutti i file di WordPress.

Come attivare WordPress in laboratorio

  1. Scaricare dal NAS del professore il file v2-xampp.zip e salvalo nel desktop.
  2. Unzippare (scompattare / estrarre) il file v2-xampp.zip sul desktop.
  3. Alla fine dell'estrazione del file aprire la "cartella" Xampp e trovare il file setup_xampp.bat
  4. Eseguire il file facendo doppio click e confermare l'avviso premendo il tasto 1 e invio.
    Attendere che il programma finisce la configurazione.
  5. Quando la configurazione finisce appare una scritta:
    Premere un tasto per continuare . . .
    Cliccare un qualsiasi tasto della tastiera per chiudere la finestra.
  6. Tornare nella "cartella" Xampp ed eseguire il file xampp-control.exe
  7. Avviare i servizi Apache e MySQL.
  8. Aprire un browser (esempio Google Chrome) ed andare sulla pagina locale di WordPress usando questo indirizzo.
    9. http://localhost:8081/wp-admin/
  9. Per accedere al pannello di WordPress inserire:
    10. Nome utente: admin
